# zhaochong_page

Personal homepage for **Zhaochong An**, restyled to match [Weiming Ren](https://cs.uwaterloo.ca/~w2ren/) (Bulma + publication cards layout, adapted from [keunhong.github.io](https://github.com/keunhong/keunhong.github.io)).

## Preview

Open `index.html` in a browser, or:

```bash
cd /Users/zhaoan/code/zhaochong_page
python3 -m http.server 8000
# then visit http://localhost:8000
```
